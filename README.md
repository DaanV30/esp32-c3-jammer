## Wiring Diagram

| ESP32-C3 | NRF24 |
|---------|-------|
| 3V3     | VCC   |
| GND     | GND   |
| GPIO 20      | CE    |
| GPIO 21      | CNS   |
| GPIO 4      | SCK  |
| GPIO 6      | MOSI  |
| GPIO 5      | MISO   |


## Required Libraries In Arduino IDE

- Arduino
- SPI
- nRF24L01
- RF24
